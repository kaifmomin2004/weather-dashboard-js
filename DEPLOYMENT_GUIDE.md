## Step 1: Create GitHub Repo

1.  Go to [GitHub.com/new](https://github.com/new).
2.  Name: `weather-dashboard-js`.
3.  Public: Yes.
4.  Create Repository.

## Step 2: Push Code

Run these commands in your `project_weather` terminal:

```bash
git remote add origin https://github.com/YOUR_USERNAME/weather-dashboard-js.git
git branch -M main
git push -u origin main
```

## Step 3: Netlify

1.  "Import from Git".
2.  Select `weather-dashboard-js`.
3.  Deploy.

## Step 4: Resume Links

Once deployed, save the links to put in your LaTeX resume.
